Le plugin Grooveshark permet à l'utilisateur d'écouter de la musique du site internet GrooveShark via un player spécialement fait pour S.A.R.A.H. GrooveShark est un site internet de streaming musical comme deezer ou spotify, l'avantage est qu'il est accessible partout, complètement gratuit, sans publicitées et dont le catalogue est très important et très actualisé. (Les nouveaux albums et hits à la radio sont généralement disponible très vite sur GrooveShark)

Grâce à votre compte GrooveShark (complètement gratuit), vous pourrez écoutez vos playlist, vos chansons préférée, les playlist que vous suivez,... Si vous n'avez pas de compte, vous pourrez quand même écouter les chansons populaires de la journée/semaine/mois, faire des recherches par artiste/album/chanson,.. Le plugin et le player dédié à S.A.R.A.H. se charge de tout. Si vous définissez aussi le password de votre compte grooveShark dans les configurations du module, vous pourrez ajouter la chanson à l'écoute à vos chansons préférée/à l'une de vos playlist,...

Installation :

Avant de pouvoir utiliser le plugin :
  • vérifiez que le programme "GroovesharkPlayerAIR.exe" qui se trouve dans le dossier /bin s'ouvre correctement (vous devriez juste voir une fenêtre avec écrit "grooveShark". vous pouvez ensuite fermer la fenêtre.

Paramêtres :

Voici les différents paramêtres que vous pouvez changer dans la configuration du plugin :
  • login : votre login (ATTENTION, votre nom d'utilisateur, pas votre email!) grooveshark (pas obligatoire mais permet d'accéder à vos chansons préférées, vos playlist, les playlist que vous suivez,...
  • password : votre password grooveshark (pas obligatoire mais permet d'ajouter une chanson à l'écoute comme "favorite",...)(vous pouvez définir votre login sans obligatoirement définir de password, vous n'aurez juste pas accés au fonctionnalitées décrites)
  • detailedSongs (oui/true/non/false): quand vous demandez à Sarah quelle est la chanson que vous écoutez, le plugin vous donne le titre de la chanson et l'artiste. Si "detailedSongs" est à "true" ou "oui", Sarah vous dira aussi le nom de l'album
  • bigPicture (oui/true/non/false): en bêta, permet d'avoir un affichage plein-écran "a la XBMC" montrant la chanson actuelle, la prochaine,... (utile si le client dispose d'un écran,...). s'il est à "false" ou "non", le player prends la forme d'une toute petite application contenant uniquement le logo de grooveshark
  • doOnOpen (last / popular / favorites): vous permet de définir ce que fait le plugin quand vous lui dites simplement de lancer de la musique. "last" vous fera écouter la dernière action que vous avez fait, "popular" vous fera écouter les chansons populaire de la journée, "favorites" vous fera écouter vos chansons préférée (uniquement si "login" est défini)

Utilisation :

Il y a tout un tas de phrase possible pour ouvrir le player, faire des recherches,... le plus simple est de lire le fichier XML pour voir toutes les phrases possibles. Voici cependant une liste de toute les fonctionnalités disponible et une phrase correspondante
  • Sarah mets de la musique : lance le player grooveShark et fait ce qu'il est défini dans "doOnOpen"
  • Sarah ferme groovshark : ferme le player grooveShark
  • Sarah coupe la musique : mets en pause la musique
  • Sarah remets la musique : remets la musique après une pause
  • Sarah chanson suivante : mets la chanson suivante
  • Sarah chanson précédante : mets la chanson précédante
  • Sarah lance mes chansons préférées : mets vos chansons préférées
  • Sarah j'aime cette chanson : mets la chanson en écoute dans vos chansons préférées
  • Sarah c'est quoi la musique : vous donne le titre de la chanson et l'artiste (et éventuellement l'album en fonction de "detailedSongs"
  • Sarah quesque tu fais sur grouvshark : vous décris ce qui est actuellement en écoute ("votre playliste Noël"/"les meilleurs chansons de Stromae"/"les chansons populaires d'aujourd'hui",...)
  • Sarah lance les chansons populaires d'aujourd'hui : vous fera écouter les chansons populaires du jours sur grooveshark (un peu comme une radio des hits)
  • Sarah met la playlist XXX : lance une de vos playlist ou celles que vous suivez
  • Sarah met à jour les playlists : mets à jour la liste de vos playlist et des playlists que vous suivez, dans le xml de language
  • Sarah met l'album Cheese : recherche le nom de l'album via la reconnaissance de Google et lance les chansons dans l'ordres de l'album. (suivi d'autre chansons qui correspondent)
  • Sarah met l'album Cheese de Stromae : recherche le nom de l'album et de l'artiste via la reconnaissance de Google et lance les chansons dans l'ordres de l'album. (suivi d'autre chansons qui correspondent)
  • Sarah met l'artiste Stromae : recherche le nom de l'artiste via la reconnaissance de Google et lance les chansons dans l'ordres des chansons les plus populaires.

Si vous avez la moindre question, n'hésitez pas à la pause sur la page google plus de S.A.R.A.H.
